# Linux Basics Introduction - JNLAB Training

This course covers the fundamentals of Linux and bioinformatics analysis. We'll start with basic Linux commands and work our way through the complete bioinformatics pipeline.

---

## Welcome

We put together these lessons to help you learn Linux and bioinformatics step by step. Whether you're new to this or just need a refresher, you'll find practical examples and clear explanations here.

---

## Topics We'll Cover

### 1. Linux Fundamentals
- Basic Linux commands and navigation
- File manipulation and organization
- System administration basics

### 2. Environment Setup
- Installing Mamba/Conda/Anaconda
- Package management
- Creating isolated environments

### 3. Quality Control
- Understanding FASTQ format
- Phred quality scoring
- FastQC analysis and interpretation
- Generating QC reports

### 4. Data Preprocessing
- Adapter trimming and filtration
- Trim Galore and fastp tools
- Quality filtering

### 5. Genome Assembly
- De novo vs reference-based assembly
- Assembly using SPAdes/MEGAHIT
- Assessment using QUAST

### 6. Genome Analysis
- Genome alignment and mapping
- Marking duplicates with PICARD
- Variant calling using FreeBayes
- Genome annotation with Prokka
- AMR gene detection with AMRFinder

### 7. Long-Read Data
- Working with Oxford Nanopore reads
- Tools: Porechop, NanoFilt
- Mapping with minimap2
- Assembly quality with QUAST

### 8. Enrichment Analysis
- Statistical enrichment analysis
- Data visualization

### 9. Metagenomics
- Metagenomics workflow
- Classification with Kraken2
- Visualization with Krona
- Assembly with FLYE
- Binning with MetaBAT2

## Course Structure

Each chapter includes:

## Tips for Success

- Practice commands in a safe environment
- Read error messages carefully
- Use --help flag for command documentation
- Document your workflow for reproducibility
